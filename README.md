Developed a full-featured eCommerce application using Java Spring Boot following enterprise-level architecture and best practices. Implemented secure user authentication and authorization, product catalog management, shopping cart functionality, order processing, payment integration, and inventory management modules.

Designed and developed RESTful APIs for seamless communication between frontend and backend systems. Utilized Spring Security, Spring Data JPA, Hibernate, and MySQL to build a scalable and maintainable backend solution. Implemented role-based access control, JWT authentication, exception handling, logging, and API validation to ensure application security and reliability.

Applied microservices concepts, database optimization techniques, and clean code principles to enhance performance and maintainability. Built responsive and efficient workflows for customer management, product administration, order tracking, and payment processing.

Technologies Used: Java, Spring Boot, Spring Security, Spring Data JPA, Hibernate, MySQL, REST APIs, Maven, JWT, Git, Docker.
